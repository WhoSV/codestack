## Getting started

**Clone the repo**

    `$ git clone <repo>`
    `$ cd codestack`

**Install dependencies**

    `$ npm install`

**Start Run**

`$ npm start`

This will initiate the project at `http://localhost:8080`.

## Build Package for Production

For building once for production (via minification).
Builds into `public/` as `bundle.js`

`$ npm run build`
